# MLM Refacor

I am trying to refactor [MLM Lagcy](https://github.com/vivek80801/mlm-lagcy). 

At current stage of the project. I have only infrastructure code. Building `mini framework` first time.

## RUN

```sh
git clone https://github.com/vivek80801/mlm-refactor
cd mlm-refactor
cp .env.sample .env
```
Edit `.env` file with appropiate configration. make sure, database server is up and running

```sh
cd public
php -S localhost:5000
```
check `localhost:5000` in browser
go to `localhost:5000` in browser

If you have xampp 

```sh
/path/to/xampp/folder/bin/php -S localhost:5000
```
